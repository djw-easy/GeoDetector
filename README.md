# Install

不要使用Pip或者conda，直接下载`geodetector.py`文件，放置到工作目录或者`Python安装路径/Lib/site-packages`目录下即可。

另外还需安装`numpy`、`pandas`和`scipy`三个库，直接使用`pip install numpy pandas scipy`安装即可。
